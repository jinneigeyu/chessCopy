> 基于`Qt5`开发的中国象棋网络对战平台(含通讯功能)，实现了在单机或联网状态下，无论是单人还是多人，无论使用系统是否相同，均可以实现象棋游戏功能。

<br>

## 特性:

**该项目主要功能模块分为：**

①玩家与自己对战

②玩家与电脑AI对战 

③多人网络对战(可跨不同系统)

④对战计时

⑤关于作品信息

## 构建：

1. 安装 Qt （Qt >= 5.9.0）

2. 运行命令:

   ```bash
   $ git clone https://github.com/xmuli/chinessChess.git
   $ cd chinessChess
   $ qmake
   $ make
   $ ./ChineseChess
   ```


或者，你可以跳过第 2 步，直接使用 Qt Creator 进行编译和运行。
