## Build:

1. Install Qt ÔºàQt >= 5.9.0Ôºâ

2. Run the command::

   ```bash
   $ git clone https://github.com/xmuli/chinessChess.git
   $ cd chinessChess
   $ qmake
   $ make
   $ ./ChineseChess
   ```


Or you can skip the second step and use Qt Creator to compile and run.
